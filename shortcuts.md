# Arukin Glossary & Shortcuts

This document defines standard shortcuts and acronyms used across development, security audits, and agent instructions in the Arukin workspace.

| Shortcut / Acronym | Full Form | Description |
| :--- | :--- | :--- |
| **AJO / AJSO** | Authorized JavaScript Origins | Allowed source domains configured in Google Cloud Console for OAuth. |
| **ARU** | Authorized Redirect URIs | Allowed callback URLs configured in Google Cloud Console for OAuth. |
| **BE** | Backend | The Supabase and Deno Edge Function layer. |
| **CC** | Credit Card | Used primarily in Razorpay billing discussions. |
| **CF** | Cloudflare | Former hosting provider for the frontend (replaced by Firebase). |
| **DPR** | Don't Push to Remote | Guardrail rule forbidding automated git pushes without explicit authorization. |
| **FE** | Frontend | The Vite + React user dashboard interface. |
| **FB** | Firebase | The web hosting platform for the frontend dashboard (`arukin.pages.dev`). |
| **GCP** | Google Cloud Platform | The developer console managing Google APIs, OAuth consent, and credentials. |
| **GH** | GitHub | Version control host. The active user account is always `azinamotoe`. |
| **OA** | OAuth | Google Authentication protocol for secure login and scoping. |
| **PTR** | Push to Remote | The authorized step of pushing local Git commits to remote repositories. |
| **RLS** | Row-Level Security | PostgreSQL security engine ensuring users only access their own data. |
| **SB** | Supabase | The BaaS backend hosting the Postgres DB and Edge Functions. |
